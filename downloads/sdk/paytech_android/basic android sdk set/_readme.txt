Preparations required to develop on A920 series terminals:
1. Please check if the terminal is in debug mode - it should show a watermark in lower right corner of the screen.
   If the terminal is not in debug mode please contact Novelpay.
2. Install PayDroid Tool
3. Install NeptuneService_*_SIG.apk
4. Then you can develop, debug and test your application using an Android Studio in the same manner as for any Android device.

In case of any questions please contact Novelpay at android@novelpay.pl
